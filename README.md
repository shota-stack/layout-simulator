# 図面レイアウトシミュレーター

図面（PDF / 画像）を背景に、什器・設備を **実寸（mm）でドラッグ＆ドロップ配置** できるブラウザ用ツールです。
サーバー不要・単一の `index.html` だけで動作します。図面データはブラウザ内だけで処理され、外部に送信されません。

**公開URL**: Settings → Pages を有効化すると `https://<ユーザー名>.github.io/<リポジトリ名>/` で開けます。

## 主な機能

- **任意の図面に対応**：PDF・画像（PNG/JPG）をアップロードして背景に設定。
- **縮尺キャリブレーション**：図面上の2点をクリックし実寸(mm)を入力するだけで「1px＝◯mm」を確定。どんな縮尺でも正確に検証可能。
- **実寸ドラッグ配置**：mm実寸で配置。ドラッグ・回転・拡縮に追従して寸法（W×D mm / φmm）をリアルタイム表示。
- **グリッド＆スナップ**：50〜1000mmグリッドと自動吸着。回転は15°刻み吸着。
- **什器ライブラリ**：客席／厨房設備／水回り・その他／自由図形。
- **複数シート**：階ごとに切り替え、配置と縮尺を個別保持。
- **保存・復元**：JSONで保存/読込。ブラウザに自動保存。
- **PNG書き出し**。

## 使い方

1. ページを開く。
2. 「＋PDF / ＋画像」で図面を読み込む。
3. 「縮尺を設定」→ 図面上の2点をクリック → 実寸(mm)を入力。
4. 左のパーツをクリック（またはドラッグ）して配置。四隅ハンドルで回転・サイズ変更、下部パネルで数値指定。
5. 「保存(JSON)」で保存、「PNG書出し」で画像出力。

### ショートカット
`Delete`=削除 / `Ctrl/Cmd+D`=複製 / `Space+ドラッグ`=画面移動 / `ホイール`=拡大縮小
